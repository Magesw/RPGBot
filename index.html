Using the bot

To use the bot, you'll need to invite it to your Discord server. Click here to invite the bot.

Alternatively, You can join the official Discord Dungeons server here.

Commands

The Discord Dungeons bot offers a range of commands.

All of the commands are prefixed with #!, @DiscordRPG or DiscordRPG.

A list of the commands can be found by using the command #!commands in a server where DiscordRPG is enabled.
